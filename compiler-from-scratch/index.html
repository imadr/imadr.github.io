Self compiling compiler from scratch

Resources on making compilers suck ass, that includes books (dragon book, engineering a compiler...etc) and most blog post or courses on the internet.
They spend too much time explaining trivialities, like tokenizing or parsing, and not nearly enough on important stuff like optimization or generating machine code that actually runs which is the point of a fucking compiler.

So we will be making a compiler for a C-like language, the compiler will be able to compile an implementation of itself in it's language.
For that we need a language that has functions, variables, structs, arrays, ifs, loops, reading and writing files from the filesystem, that's it.

NO BULLSHIT LALR, NO VISITOR PATTERNS, NO OOP TRASH, NO VIRTUAL MACHINE BYTE CODE NONSENSE, only straightforward code, and a compiler that generates a program that runs.
SHIT AINT MAGIC, this is what we will be making in order:

  • Tokenizer: Takes a source code string as input and outputs a list of tokens (literals, keywords, punctutations like semicolons and so on)
  • Parser: Takes the list of tokens and turns it into an abstract syntax tree (AST), checks if the source code syntax is correct. Simple recursive descent parser
  • Semantics: Type checking, name checking
  • Optimization: Peephole, deadcode elimination
  • Code generation: Generating x86-64 (or x64, or amd64 whatever) assembly, register allocation

👉 at this point, you can take the generated assembly and use an assembler like FASM to generate an executable, but that's for the weak...

  • Assembly and linking: Turn into machine code and create an executable that can run on windows (PE format) and linux (ELF format)

The main program

This is what our compiler main should look like : 
int main() {
    str file_path = "test.x";
    pair file_content = read_whole_file(file_path);
    if (file_content.first) {
        printf(file_content.first);
        return 1;
    }

    TokenizerState tokenizer_state = tokenize(file_content.second, file_path);
    if (tokenizer_state.error) {
        printf(tokenizer_state.error);
        return 1;
    }

    SemanticState semantic_state;
    init_semantics(&semantic_state);

    ParserState parser_state = parse(tokenizer_state.tokens, &semantic_state, file_path);

    CodegenState codegen_state = codegen(parser_state.root_node);

    return 0;
}
