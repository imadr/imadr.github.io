Cross-platform window in C

Motivations:
  • I want complete control over my program
  • It's a fun learning experience
  • GLFW is nice, but I can't figure out how to statically link it
Building the program for multiple platforms should not be any more complicated than this: 
gcc.exe main.c -l gdi32 -mwindows
build_windows.bat
 
#!/bin/bash
gcc main.c -l X11 -D X11
build_linux_x11.sh
 
#!/bin/bash
gcc main.c -l wayland-client -D WAYLAND
build_linux_wayland.sh
